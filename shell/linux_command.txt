pwd
print work directory
現在作業中のディレクトリを見る

cd
change directory
行きたいディレクトリに移動する

ls (ls -la)
list
現在作業中のディレクトリの内容を見る
-la : -l(long:詳細な内容) + -a(all:隠したものを含めて)
ls -la
現在作業中のディレクトリの隠したものを含め全てのファイルとディレクトリを見る

mkdir
make directory
新しいディレクトリを作る

touch
ファイルやディレクトリの最近アップデート日時を現在の時間に変える

rm
remove
ファイルやディレクトリを削除する

mv
move
ファイルやディレクトリを動かす
