git command

clone
リモートリポジトリの内容をローカルリポジトリにダウンロードする

branch
新しいブランチを生成する

checkout
他のブランチに乗り換える

log
commitした記録を確認する

status
リポジトリ内の変更事項を確認する

add
ワークツリーで作業したものをステージに上げる

commit
変更事項をローカルリポジトリに適用する

diff
ファイルのどの部分が変更されたか確認する

push
commitいたファイルをリモートリポジトリにアップロード

pull
リモートリポジトリの内容をローカルリポジトリに適用する

fetch
リモートリポジトリの変更事項を確認する

merge
現在のブランチを基準にほかのブランチを併合し、変更事項を適用する
